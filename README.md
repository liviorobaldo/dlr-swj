This GitHub repository contains the Java source code accompanying the paper "Deontic Logic Reified: a deontic logic for the Semantic Web". The logic and all examples are implemented in RDF and SPARQL, and make use of the <a href="https://jena.apache.org/">Apache Jena</a> libraries for execution.

There are two main subfolders: `DLRsuite` and `SHACLreasonerFromRobaldoetal2023`. The RDF and SPARQL resources defining Deontic Logic Reified (DLR) are contained in `DLRsuite`, while `SHACLreasonerFromRobaldoetal2023` contains the SHACL-based reasoner from the paper <a href="https://link.springer.com/article/10.1007/s10506-023-09360-z">[Robaldo et al., 2023]</a>, adapted to operate on the same DLR examples for comparison purposes.
